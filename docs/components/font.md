# Font 字体

## 概述

Qnet UI 的 CSS 对字体进行了统一规范，力求在不同平台、浏览器下能显示出其最佳的效果。我们推荐 macOS（iOS）优先的策略，在不支持苹方字体的情况，使用备用字体。
